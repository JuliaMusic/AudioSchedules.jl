# AudioSchedules

[![Latest](https://img.shields.io/badge/docs-dev-blue.svg)](https://JuliaMusic.github.io/AudioSchedules.jl/dev)
[![CodeCov](https://codecov.io/gh/JuliaMusic/AudioSchedules.jl/branch/master/graph/badge.svg)](https://codecov.io/gh/JuliaMusic/AudioSchedules.jl)

AudioSchedules allows you to schedule a sequence of overlapping audio synthesizers.
Critically, it allows you to do this without lags or gaps. See the documentation for an
example.
